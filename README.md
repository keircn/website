## Prerequisites

- Node.js
- Package manager (pnpm/npm)

## Quick start

```bash
# install dependencies
pnpm i

# start development server
pnpm run dev
```

## Build & Production

Build the optimized production bundle:

```bash
pnpm run build
pnpm run start
```
